# COMP2090SEF/8090SEF Course Project (Template Implementation)

This repository contains **both Task 1 and Task 2** under a single repo structure.

- `task1/` — **Student Assignment Manager** (OOP-based application)
- `task2/` — **Heap (data structure) + Heap Sort (algorithm)** (self-study)
- `reports/` — editable report drafts (DOCX)
- `slides/` — presentation deck (PPTX)

> ⚠️ Academic honesty reminder: this is a **starter implementation**. You must review, understand, and personalize it (names, student IDs, contribution, reflection, test cases, etc.) before submission.

