# Task 1 — User Guide (Student Assignment Manager)

## Requirements
- Python 3.10+ recommended
- No third‑party packages required

## How to run
From the repo root:

```bash
python -m task1.main
```

The program creates/uses a local JSON database file:
- `task1_data.json` (created automatically next to the code)

## Main features
1. Add task (assignment/exam/general)
2. List all tasks
3. Search by keyword
4. Filter by course
5. Mark complete / uncomplete
6. Sort by due date
7. Save & exit

## Notes
- Dates are entered in `YYYY-MM-DD` format.
- IDs are auto-generated.
